Notes on switching to plaintext

Why I left Notion (and what I kept)

Why

I spent six months in Notion before realizing my notes were trapped. Migrating out took a weekend of janky exports and reformatting. Never again.

If you take only one thing from this post: own your text files.

The shape of the problem isn't "Notion is bad" — it's that any tool that holds your prose hostage to its UI eventually becomes a tax on thinking. The fix is to write in formats you can grep, version, and move.

What changed

Worth flagging upfront: this isn't a clean migration. I still use Google Docs for collaboration and Notion for shared databases. The shift is where the canonical copy lives — which, for me, is now a folder of markdown files in a git repo1.

Three things made it stick: a daily ritual, a sync pipeline, and the realization that almost nothing needs to be in a database.

  1. …and yes, the irony of that sentence appearing on a site whose own pipeline runs through Google Docs is not lost on me. ↩

# x styling

## Global

Personal essay. Lean literary, not techy. Warm cream background, deep brown text,
soft gold accent. Body in a comfortable serif (Iowan Old Style / Charter), max
width 65ch, generous line height. Slightly narrower on mobile.

## Tags

**pull-quote**
Large serif, around 1.4× body size. Italic. No quotation-mark characters (the
weight is the quote). Soft gold horizontal rule a few rems below.

**callout**
Cream-yellow box (warmer than the page background), rounded corners, generous
padding, slightly smaller font than body. Italic optional. Used for asides,
caveats, and "before we begin" framings.

**footnote-ref**
Subtle superscript link. Same gold as the accents. Don't make it pop.

## Page-level

Centered title block, eyebrow above title in small caps, subtitle in regular
weight underneath. H2s sit a couple of lines below the previous paragraph
without a decorative rule.

pipeline.yml GitHub Actions, daily 09:03 UTC
name: docs-to-blog sync
on:
  schedule: [{ cron: "3 9 * * *" }]
  workflow_dispatch:

jobs:
  sync:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Restore gdoc OAuth token
        run: |
          mkdir -p ~/.config/gdoc
          echo "$GDOC_TOKEN_JSON_B64" | base64 -d > ~/.config/gdoc/token.json
        env: { GDOC_TOKEN_JSON_B64: ${{ secrets.GDOC_TOKEN_JSON_B64 }} }

      - name: Detect change via Drive version
        id: probe
        run: scripts/probe-version.py

      - name: Pull doc body + x styling tab
        if: steps.probe.outputs.changed == 'true'
        run: |
          gdoc cat "$DOC_URL" > content/main.md
          scripts/pull-styling-tab.py "$DOC_URL" > content/_styling.md

      - name: Fuzzy reconcile anchors
        if: steps.probe.outputs.changed == 'true'
        id: reconcile
        run: scripts/reconcile.js  # diff-match-patch, writes orphans.json

      - name: Claude reconciler (only on orphans)
        if: steps.reconcile.outputs.orphans > 0
        run: |
          claude -p --bare --max-turns 6 \
            --allowedTools "Read,Edit,Bash(git:*)" \
            "$(cat scripts/reconcile-prompt.md)"
        env: { ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }} }

      - name: Commit & open PR if anchors changed
        run: scripts/open-pr-if-dirty.sh
